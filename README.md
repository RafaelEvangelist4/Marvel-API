# Marvel-API

 **SPA usando API da Marvel**
 
 Ferramentas usadas na criação da SPA:
 - ReactJS
 - HTML5
 - CSS3
 - Javascript
 
 ## Rodando o projeto:
 no diretorio do projeto (..cd marvel), abra o cmd e digite: 
 
 >"yarn start"
 
 No browser entre na porta localhost:3000 (http://localhost:3000)
